# Development

## Build

### Requirements libxml

On Linux:

```bash
apt-get install libxml2-dev
```

On macOS:

```bash
brew intall libxml2
```

### Build

On Linux:

```bash
tools/build-linux.sh
```

On macOS:

```bash
tools/build-darwin.sh
```

## test

```bash
tools/test.sh
```
